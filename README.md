# chatbot_ClariceLispector
Projeto de chatbot desenvolvido na aula de PLN, com o objetivo de utilizar o corpus das obras de um autor, realizar a limpeza, o treinamento da máquina e avaliação das métricas para que quando solicitado a máquina retorne um poema escrito com o mesmo estilo da autora escolhido (Clarice Lispector).
